# 理カード（RiCard）

毎朝、生年月日を入れると、東洋占術＋「理」の視点で「今日の流れ」を
美しい1枚のカードにして返すアプリ。**当てて煽らない。淡々と整える。**

世界の数千万人に届く一発を狙うプロジェクトの第1弾。
（経緯と方針はメモリ `project_app-idea-research-2026-06-18.md` を参照）

## 進捗
- [x] **Day 1-2 占術エンジン** … 生年月日＋対象日 → 八字・九星・五行の関係 → 「今日の流れ」の素材
- [x] **Day 3-4 理の声** … 断定/煽りなしの淡々とした文章＋禁止表現チェッカー（`engine/voice.py`, `style_guard.py`, `STYLE.md`）

## セットアップ
```
pip install -r requirements.txt
```
（`lunar_python` は純Python。コンパイル不要。干支・節気・九星をローカル計算）

## 動かす
```
# Webアプリ（本体）
python app.py
# → ブラウザで http://127.0.0.1:5390/

# 単体確認
python demo.py        # 素材
python demo_card.py   # 整った文章＋禁止表現チェック
python gen_cards.py   # カード画像3スタイルを data/ に生成
```

## 設計メモ
- **すべてローカル計算・外部送信なし。** 生年月日は端末内で命式（抽象データ）に変換し、文章化AIには配置情報だけを渡す方針（次工程）。
- **誠実さが堀。** 断定/煽り禁止。画面に「娯楽・自己内省のため／当てるものではない」を固定表示。
- **簡略化している点（正直に）**：九星の方位は「その日の九星の定位」を参考として出す簡易版。本格的な吉方位（年盤・月盤・日盤の同会判定）は未実装。文章化工程で「参考」と明示する。
- 占術エンジンは 理カード だけでなく、後続の弾（縁の相性／守護精霊／前世物語）でも使い回す前提。
