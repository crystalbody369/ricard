# 理カードの声（文章ルール）

理カードの文章は「当てる占い」ではなく「**淡々と整えるための言葉**」。
ここがこのアプリの**誠実さ＝堀**。崩したら別物になる。

## 3つの原則
1. **断定しない**… 「必ず」「絶対」「確実に」は使わない。「〜とよいでしょう」「〜かもしれません」「〜しやすい日」で止める。
2. **煽らない・脅さない**… 不安・恐怖・危険・呪い・損で人を動かさない。
3. **淡々と流れる**… 押すと止まる／整えると進む。命令形で急かさない。静けさを残す。

## 声のトーン（一神会「理」の世界観）
- 上から言わない。隣で静かに見ている人の口調。
- 良い日も悪い日も「流れの一部」として淡々と。良い日を煽らない、悪い日を脅さない。
- 余白を残す。言いすぎない。最後はそっと閉じる（例:「止まって見えるのは、整えている時間。」）

## カードの構成（5つ）
1. 今日の理（一言の枠づけ） … 例「今日は、静かに整える日。」
2. 流れ … その日の進み方
3. 縁・人 … 人間関係の動き
4. 動き … 方位・タイミング（※簡易計算の「参考」と明示）
5. 整える … 今日できる小さな一手、ひとつだけ

## 禁止語リスト（`engine/style_guard.py` が機械的にチェック）
- **断定**: 必ず／絶対／確実に／間違いなく／100%／保証します
- **煽り・不安**: 危険／警告／不幸／呪い／祟り／たたり／災い／最悪／地獄／破滅／終わりです
- **金運の断定**: 大儲け／必ず儲か／儲かります／金運最高／億万長者／大金が入ります
- **緊急・強制**: 今すぐ／急いで／買わないと／しないと損／今だけ／手遅れ
- **医療の断定**: 必ず治／治ります／完治します／病気が治る

→ カードは表示・投稿の前に必ず `check_text()` を通し、1件でも当たれば出さない。

## 必ず画面に出す注記
「これは娯楽・自己内省のための目安です。断定するものではありません。方位は簡易計算の参考です。」
